+ vue-cli
+ vuex
+ vue-ydui
+ axios
+ vue-router

+ ...

## 功能需求
- ✅ 推荐歌单
- ✅ 歌手查询
- ✅ 播放歌曲
- ✅ 搜索歌曲

## 使用
 1 ：git clone 

 2 ：进入到vue 下，npm install （默认安装了node）

 3 ：进入到vue 下，npm run dev

 （端口是1111，可以在vue/config/index.js 修改）
